ITYPE - An automated typing tool

Its a project aimed to digitising documents with the aspect of editing as our main goal while saving the user time taken in typing characters.

Our solution is not yet deployed to the web but instead our source code repository  is accessible here on github

Technologies employed
-Python in django
-HTML, CSS, Bootstrap
-Tesseract OCR


How to setup the project
1. Set up a django virtual eenvironment on your PC (This is optional)
2. Open any Terminal and activate the virtual environment (depends on if you created one)
3. Clone our repository to a destined folder or in a virtual env
4. Install necessary dependencies in the requirements.txt file
7. Run the server and browse the application from the host url
